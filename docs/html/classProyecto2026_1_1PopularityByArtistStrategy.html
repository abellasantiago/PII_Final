Detailed Description

Clase de tipo IRecommendationStrategy que recomienda música del mismo artista. Cumplimiento Expert: la estrategia conoce cómo buscar contenido relacionado por artista.

Se usa para recomendar contenido relacionado a una música base. Cumplimiento OCP y patrón Strategy: agrega un criterio de recomendación sin cambiar el motor.
